# ¿Qué es SUMA?

SUMA (Sistema Unificado de Modelado y Artefactos) es un paradigma de programación que pone el énfasis en la trazabilidad, la claridad y la mantenibilidad del software.

## Principios Fundamentales

### 1. Todo es un Artefacto
En SUMA, todo elemento del sistema es un artefacto con trazabilidad completa.

### 2. Modelos como Ciudadanos de Primera Clase
Los modelos no son meras estructuras de datos, sino entidades vivas del sistema.

### 3. Dominios Explícitos
Los límites entre dominios son explícitos y respetados.

### 4. Queries como Lenguaje Universal
Las queries son el lenguaje universal para consultar y transformar artefactos.

## Por qué SUMA

El desarrollo de software tradicional sufre de problemas como:
- Falta de trazabilidad
- Acoplamiento implícito
- Dificultad para entender el impacto de los cambios
- Documentación desincronizada

SUMA aborda estos problemas de raíz.
